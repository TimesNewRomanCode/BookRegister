## Laravel-API

## Иструкция

### 1:

Создать .env и скопировать все из .env.example

### 2:

Установать composer

```
composer install
```


### 3:

Провести миграции

```
php artisan migrate
```

### 4:

Запуск

```
php artisan serve
```

### 5:

Для проверки работы используйте свагер

```
http://127.0.0.1:8000/docs
```
